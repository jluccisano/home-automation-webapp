# Home Automation WebApp

[![tests][tests]][tests-url]

## Docker

### Build

```bash
docker build -t jluccisano/home-automation-webapp .
```

### Run

```bash
docker run -dit --name home-automation-webapp \
    -p 8515:8515 \
    jluccisano/home-automation-webapp
```
